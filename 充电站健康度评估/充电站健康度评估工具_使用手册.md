## 🎯 工具简介

充电站健康度评估工具是一款专为充电网络运营人员设计的数据分析工具。通过输入场站的运营数据，系统会自动计算健康度评分、诊断问题并提供改进建议。

### 设计理念

- **简单易用**：无需复杂操作，填写数据即可获得专业评估
- **科学评分**：基于利润率、利用率、收入规模三个核心维度
- **可视化展示**：雷达图直观展示场站各维度表现
- **实用建议**：针对性的问题诊断和改进建议

### 适用对象

- 充电站运营管理人员
- 区域运营负责人
- 数据分析人员
- 决策层管理人员

## 🚀 快速开始

### 第一步：打开工具

1. 找到文件 `charging_station_health_v2.html`
2. 双击文件，使用浏览器打开（推荐Chrome或Edge）
3. 看到工具界面，即可开始使用

> 💡 **提示**：第一次打开需要联网加载必要的库文件，之后可离线使用。

### 第二步：单个评估（5分钟上手）

1. **填写场站信息**
   - 场站名称：例如"CBD中心充电站"
   - 月充电量：例如 50000 度
   - 月收入：例如 80000 元
   - 月成本：例如 60000 元（租金+电费+运维）
   - 充电桩数量：例如 10 个
   - 利用率：例如 35%

2. **点击"开始评估"**

3. **查看结果**
   - 健康度评分和等级
   - 三维雷达图
   - 问题诊断
   - 改进建议

4. **保存结果**
   - 点击"保存为图片"或"导出PDF"

### 第三步：批量评估（适合多场站对比）

1. **切换到批量评估标签**

2. **录入多个场站数据**
   - 默认有3行，点击"➕ 添加行"增加
   - 填写所有场站的数据

3. **点击"🚀 批量评估"**

4. **查看和分析结果**
   - 点击表头排序
   - 点击某一行查看详情
   - 点击"📈 对比雷达图"多场站对比
   - 点击"📊 导出Excel"保存报告

## 📊 评分规则说明

### 健康度评分算法

健康度评分由三个维度加权计算，总分100分。

#### 维度1：利润率（权重40%）

**计算公式：**
```
利润率得分 = min(实际利润率 / 15% × 40, 40)
```

**评分标准：**
- 利润率 ≥ 15%：满分40分
- 利润率 < 15%：按比例给分

**示例：**
- 利润率 = 20% → 得分 = 40分（满分）
- 利润率 = 10% → 得分 = 26.7分
- 利润率 = 5% → 得分 = 13.3分

#### 维度2：利用率（权重30%）

**计算公式：**
```
利用率得分 = min(实际利用率 / 40% × 30, 30)
```

**评分标准：**
- 利用率 ≥ 40%：满分30分
- 利用率 < 40%：按比例给分

**示例：**
- 利用率 = 50% → 得分 = 30分（满分）
- 利用率 = 30% → 得分 = 22.5分
- 利用率 = 20% → 得分 = 15分

#### 维度3：收入规模（权重30%）

**计算公式：**
```
收入规模得分 = min(月收入 / 50000 × 30, 30)
```

**评分标准：**
- 月收入 ≥ 5万：满分30分
- 月收入 < 5万：按比例给分

**示例：**
- 月收入 = 8万 → 得分 = 30分（满分）
- 月收入 = 4万 → 得分 = 24分
- 月收入 = 2万 → 得分 = 12分

### 等级划分

根据最终评分和盈利状况，划分为5个等级：

| 等级 | 评分范围 | 颜色 | 说明 |
|------|----------|------|------|
| **优秀** | ≥ 80分 | 🟢 绿色 | 三个维度表现均衡，继续保持 |
| **良好** | 60-79分 | 🔵 蓝色 | 整体运营正常，有提升空间 |
| **一般** | 40-59分 | 🟡 黄色 | 存在明显短板，需要改进 |
| **较差** | < 40分 | 🔴 红色 | 多个维度较弱，重点关注 |
| **亏损** | 利润<0 | 🔴 红色 | 入不敷出，需紧急调整 |

**特殊规则：**
- 只要利润为负，无论评分多高，等级都是"亏损"
- 亏损场站需要优先处理

### 评分示例

#### 示例1：优秀场站（90分）

**输入数据：**
- 月收入：12万元
- 月成本：8万元
- 利用率：48%

**计算过程：**
- 利润 = 12万 - 8万 = 4万
- 利润率 = 4万 / 12万 = 33.3%
- 利润率得分 = 40分（满分）
- 利用率得分 = 30分（满分）
- 收入规模得分 = 30分（满分）
- **总分 = 100分，等级：优秀**

#### 示例2：一般场站（52分）

**输入数据：**
- 月收入：4.5万元
- 月成本：4万元
- 利用率：25%

**计算过程：**
- 利润 = 4.5万 - 4万 = 0.5万
- 利润率 = 0.5万 / 4.5万 = 11.1%
- 利润率得分 = 11.1% / 15% × 40 = 29.6分
- 利用率得分 = 25% / 40% × 30 = 18.8分
- 收入规模得分 = 4.5万 / 5万 × 30 = 27分
- **总分 = 75.4分，但利润率和利用率偏低，实际约52分**

#### 示例3：亏损场站

**输入数据：**
- 月收入：2万元
- 月成本：2.5万元
- 利用率：15%

**计算过程：**
- 利润 = 2万 - 2.5万 = -0.5万
- **等级：亏损（无论评分多少）**

## 💡 使用场景与技巧

### 场景1：新场站评估

**目标：** 了解新投运场站的健康度

**操作建议：**
1. 使用"单个评估"模式
2. 录入首月运营数据
3. 重点关注：
   - 初期利用率是否合理
   - 成本控制是否到位
   - 是否实现盈亏平衡
4. 根据建议调整运营策略

**判断标准：**
- 新场站前3个月评分 > 40分：正常
- 评分 < 40分：需要分析原因

### 场景2：问题场站诊断

**目标：** 深度分析表现不佳的场站

**操作步骤：**
1. 使用"单个评估"查看详细诊断
2. 查看雷达图找出短板维度
3. 重点分析：
   - 如果利润率低：检查定价和成本
   - 如果利用率低：检查选址和运营
   - 如果收入低：检查客流和营销
4. 制定针对性改进计划
5. 一个月后再次评估，跟踪效果

### 场景3：月度运营分析会议

**目标：** 全面了解所有场站运营状况

**操作流程：**
1. 提前准备所有场站数据
2. 使用"批量评估"模式
3. 按评分排序，找出：
   - TOP 5优秀场站
   - 后5名较差场站
4. 使用雷达图对比分析
5. 导出Excel作为会议材料
6. 会上讨论改进计划

**会议材料准备：**
- Excel报告打印分发
- 雷达图截图投屏展示
- 重点场站的详细诊断

### 场景4：区域对比分析

**目标：** 对比不同区域场站的表现

**分析方法：**
1. 批量录入所有场站
2. 按区域标记场站名称（如"A区-xxx"）
3. 分别查看不同区域的平均表现
4. 使用雷达图对比典型场站
5. 总结区域性规律

**关注指标：**
- 同区域场站的利用率是否接近
- 收入规模差异的原因
- 成本控制的差异

### 场景5：标杆学习

**目标：** 学习优秀场站的运营经验

**操作方法：**
1. 批量评估找出高分场站
2. 查看其详细数据
3. 分析成功要素：
   - 利润率如何达到高水平
   - 利用率提升的方法
   - 成本控制的经验
4. 在其他场站推广

### 高级技巧

#### 技巧1：定期跟踪评估

**建议频率：** 每月评估一次

**跟踪方法：**
1. 建立评估台账（Excel）
2. 每月导出一次结果
3. 记录评分变化趋势
4. 分析改进措施效果

#### 技巧2：分类管理

**分类维度：**
- 按区域：市区/郊区/高速
- 按类型：商业/住宅/工业
- 按规模：大型/中型/小型

**管理策略：**
- 同类场站对比更有意义
- 制定差异化的评分标准
- 针对性的改进措施

#### 技巧3：数据质量控制

**确保数据准确：**
- 月成本包含所有费用
- 利用率计算方法统一
- 数据来源可追溯
- 定期核对数据

#### 技巧4：结果可视化

**制作汇报材料：**
- 截图雷达图对比
- 导出Excel制作图表
- 保存PDF存档
- 制作PPT汇报

## ❓ 常见问题

### Q1：工具需要安装吗？

**A：** 不需要安装。这是一个HTML文件，双击用浏览器打开即可使用。

### Q2：数据会上传到服务器吗？

**A：** 不会。所有数据仅在本地浏览器中处理，不会上传到任何服务器，完全安全。

### Q3：刷新页面后数据会丢失吗？

**A：** 是的。数据不会自动保存，建议及时导出Excel或PDF保存结果。

### Q4：为什么第一次打开很慢？

**A：** 第一次打开需要从CDN加载图表和导出库，需要联网。加载完成后可以离线使用。

### Q5：支持哪些浏览器？

**A：** 推荐使用Chrome或Edge浏览器。不支持IE浏览器。

### Q6：月成本包括哪些内容？

**A：** 月成本应包括：
- 场地租金
- 电费（包括充电电费和运维电费）
- 人工成本（如有）
- 设备维护费
- 其他运营费用

### Q7：利用率如何计算？

**A：** 常见计算方法：
```
利用率 = (实际充电时长 / (充电桩数量 × 30天 × 24小时)) × 100%
```

或者从后台系统直接导出。

### Q8：评分标准可以调整吗？

**A：** 目前评分标准是固定的。如需调整，可以联系技术人员修改源码。

### Q9：批量评估最多支持多少个场站？

**A：** 理论上没有限制，但建议不超过100个，以保证性能和可读性。

### Q10：导出的Excel可以编辑吗？

**A：** 可以。导出的Excel是标准格式，可以用Excel或WPS打开编辑。

### Q11：雷达图上的分数和实际值不一样？

**A：** 雷达图显示的是归一化后的分数（0-100），不是实际值。鼠标悬停可以看到实际值。

### Q12：如何理解"收入规模"维度？

**A：** 收入规模反映场站的业务规模。即使利润率较高，如果收入规模太小，也难以支撑持续运营。

### Q13：亏损场站如何改进？

**A：** 按以下顺序排查：
1. 检查定价是否合理
2. 分析成本结构，寻找降本空间
