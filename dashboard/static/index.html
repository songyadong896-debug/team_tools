Excel模板说明

字段名 必填 说明 示例
线路名称 是 线路的名字 北京一日游
序号 是 1到n的顺序号，决定导航顺序（仅沿途点） 1, 2, 3...
CZID（非必填） 否 保留字段，可留空 -
方向（非必填） 否 保留字段，可留空 -
城市 是 地点所在城市 北京
地点名称 是 具体地点名称 天安门广场
地点类型 是 "沿途"参与导航，"景点"和"场站"仅显示 沿途/景点/场站

操作说明

  1. 下载Excel模板文件
  2. 按照模板格式填写线路信息
  3. 上传Excel文件，系统自动匹配POI
  4. 确认导入后，可在地图上查看线路
  5. 左键点击点位查看详情，右键点击编辑（仅编辑模式）
