# LCP Business Management



## Getting started

### 开发
项目使用python 3.9 + 开发调试通过

### 本地调试
1. （必选）调整database.py 里的db内容
2. （可选）调整本地API 端口，默认8080
3. 访问 http://127.0.0.1:8080/ 可以看到返回 hello world JSON数据

### 测试环境
1. 测试环境部署了融合云，融合云租户是**商业系统-服务与充电系统-充电网络**。域名是：https://lcp-business-management-test.inner.chj.cloud/
