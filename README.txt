充电站健康度评估工具 - 快速指南
========================================

📦 文件说明
-----------
本工具包含以下文件：
1. charging_station_health_v2.html  - 评估工具（主文件）
2. 充电站健康度评估工具_使用手册.md  - 详细使用手册
3. README.txt  - 本文件（快速指南）


🚀 快速开始（3步上手）
-----------------------
第一步：打开工具
  - 双击 charging_station_health_v2.html 文件
  - 用浏览器打开（推荐Chrome或Edge）
  - 首次打开需要联网（加载必要组件）

第二步：选择模式
  - 单个评估：快速评估单个场站
  - 批量评估：一次性评估多个场站

第三步：录入数据
  - 填写场站信息（名称、充电量、收入、成本、桩数、利用率）
  - 点击"开始评估"或"批量评估"按钮
  - 查看结果和建议


✨ 主要功能
-----------
√ 健康度评分（0-100分）
√ 雷达图可视化分析
√ 问题自动诊断
√ 改进建议生成
√ 批量对比排序
√ Excel表格导出
√ PDF报告导出


📊 评分标准（速记）
-------------------
总分100分 = 利润率40分 + 利用率30分 + 收入规模30分

- 利润率：15%及以上得满分（40分）
- 利用率：40%及以上得满分（30分）
- 收入规模：5万元/月及以上得满分（30分）

等级划分：
  优秀：80分以上（绿色）
  良好：60-79分（蓝色）
  一般：40-59分（黄色）
  较差：40分以下（红色）
  亏损：利润为负（红色）


💡 使用技巧
-----------
1. 批量评估时，点击表头可以排序
2. 点击表格中的某一行，查看详细诊断
3. 使用雷达图对比多个场站的优劣势
4. 及时导出Excel或PDF保存结果（刷新页面会丢失数据）
5. 点击右下角"❓"按钮查看详细帮助


📝 数据录入说明
---------------
必填字段：
  场站名称：例如"CBD中心充电站"
  月充电量：单位"度"，例如 50000
  月收入：单位"元"，例如 80000
  月成本：单位"元"，包括租金+电费+运维，例如 60000
  充电桩数量：例如 10
  利用率：0-100之间，例如 35

注意：月成本 = 租金 + 电费 + 运维费用 + 其他成本


⚠️ 重要提示
------------
1. 本工具为纯前端应用，数据不会上传到服务器
2. 刷新页面后数据会丢失，记得及时导出
3. 首次使用需要联网加载库文件
4. 推荐使用Chrome或Edge浏览器（不支持IE）
5. 导出的Excel文件包含完整数据和诊断建议


🔧 常见问题
-----------
Q: 为什么打开很慢？
A: 第一次打开需要从CDN加载组件库，需要联网。之后可以离线使用。

Q: 数据会保存吗？
A: 不会自动保存，请及时导出Excel或PDF。

Q: 可以编辑导出的Excel吗？
A: 可以，导出的是标准Excel格式。

Q: 雷达图上的数字是什么意思？
A: 雷达图显示的是归一化后的得分（0-100），鼠标悬停可查看实际值。

Q: 批量评估最多支持多少个场站？
A: 理论上无限制，但建议不超过100个以保证性能。
